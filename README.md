# HTML STUDY
## Оформление текста
```css
selector {
	/* 'em' задаёт размер относительно родительского */
	font-size: ...; /* Задаёт размер шрифта */ /* inheritable */

	line-height: ...; /* Задаёт межстрочный интервал */

	/* sans-serif - шрифт без засечек */
	/* serif - шрифт с засечками */
	/* monospace - моноширинный шрифт */
	/* cursive - шрифт с неформальным начертанием */
	/* fantasy - декоративный шрифт */
	font-family: "PT Sans", "Arial", sans-serif; /* Задаёт семейство шрифта */

	/* bolder - жирнее, чем текущее или унаследованное значение */
	/* lighter - тоньше, чем текущее или унаследованное значение */
	font-weight: ...; /* Задаёт насыщенность шрифта */

	/* left, right, center */
	/* justify - по ширине блока. Текст растягивается */
	text-align: ...; /* Горизонтальное выравнивание inline элементов */

	/* top, middle, bottom, baseline, px, %, ... */
	/* Задаётся для самого выравниваемого элемента */
	vertical-align: ...; /* Вертикальное выравнивание */
}
```
